# 🛍️ LUXE STORE — Premium E-Commerce Website

LUXE STORE is a modern, premium e-commerce web application designed to deliver a luxury shopping experience. It features a sleek UI, smooth navigation, and interactive components like cart, wishlist, search, and product filtering.

---

## ✨ Features

* 🏠 **Multi-Page Navigation**

  * Home, Shop, Product Detail, Wishlist, Checkout, About

* 🔍 **Smart Search**

  * Real-time product search with dynamic results

* 🛒 **Shopping Cart**

  * Add/remove items
  * Live cart updates
  * Checkout system

* ❤️ **Wishlist**

  * Save favorite products
  * Manage wishlist items easily

* 📂 **Category Filtering**

  * Electronics, Fashion, Home & Living, Sports

* 🔃 **Sorting Options**

  * Price (Low → High / High → Low)
  * Best Rated
  * Name A-Z

* 📱 **Responsive Design**

  * Mobile-friendly with hamburger menu

* 🎨 **Modern UI/UX**

  * Custom cursor effects
  * Toast notifications
  * Smooth animations

* 📧 **Newsletter Subscription**

  * Email input for updates & offers

---

## 🧱 Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling (External: `styles.css`)
* **JavaScript (Vanilla JS)** – Functionality (`logic.js`)
* **Font Awesome** – Icons
* **Google Fonts** – Typography

---

## 🚀 How to Run

1. Download or clone the repository

2. Make sure all files are in the same folder:

   * `index.html`
   * `styles.css`
   * `logic.js`

3. Open `index.html` in your browser

👉 No server required (runs locally)

---

## 🧩 Key Functionalities

* `navigate(page)` → Switch between pages
* `filterCategory(category)` → Filter products
* `openCart()` → Open cart sidebar
* `openSearch()` → Open search modal
* `handleSearch(query)` → Live search
* `subscribeNewsletter()` → Newsletter feature
